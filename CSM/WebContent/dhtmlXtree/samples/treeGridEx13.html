Javascript Tree Grid Extension. Action handlers
 
<div id="treeBox" style="width:200;height:200"></div> <script> treeGrid=new dhtmlXTreeGridObject(document.getElementById('treeBox'),"100%","100%",0); ... treeGrid.tree.setDragHandler(onDrop);//set function object to call on drop treeGrid.tree.setOnClickHandler(onNodeSelect);//set function object to call on node select treeGrid.tree.setOnOpenHandler(aFunc);//set function to call on open/close node treeGrid.tree.setOnCheckHandler(aFunc);//set function to call on open/close node treeGrid.tree.setOnDblClickHandler(aFunc);//set function to call on dbl click </script>
  • Selected node ID will be passed to function specified as argument for setDefaultAction(funcObj)
  • Dropped node ID and new parent node ID will be passed to function specified as argument for setDragFunction(funcObj)
  • node ID will be passed to the function specified as argument for setOpenAction(aFunc)
  • node ID will be passed to the function specified as argument for setDblClickAction(aFunc)
